# [Preparação de Dados para Modelos de aplicação - Resumo]

 Preparação de Dados para Modelos de aplicação criado em [C#](http://startbootstrap.com/), [Excel](https://docs.microsoft.com/pt-br/dotnet/csharp/) e [SqlServer](https://www.microsoft.com/pt-br/sql-server/sql-server-2019)
 
 
 Este tema apresenta o código da aplicação para realizar a importação de dados de uma arquivo excel extraído do site da [Bolsa de Valores](http://www.b3.com.br/pt_br/).

## Uso

### Uso Básico

Após o download, deve executar o script de criação do Banco de Dados seguindo os tipos especificados na imagem que contém no arquivo.
Realizar também a alteração da linha de conexão com o banco de dados no código, trocando o nome do computador para o hostname do seu micro.
Estes são as atenções devidas que você deve tomar antes de executar o programa.

### Uso Avançado

Após o programa ser executado no Visual Studio, abrirá a janela para escolher o arquivo a ser importado, este arquivo excel deve ser interpretado de acordo com as normas do site da B3. (http://www.bmfbovespa.com.br/pt_br/servicos/market-data/historico/mercado-a-vista/cotacoes-historicas/)
Após essa interpretação basta anexar o arquivo com o BUTTOM disponível, e clicar em "import".

## Sobre
Atualmente, existem muitas possibilidades de investimento no mercado brasileiro. Também existem muitos modelos de otimização e algoritmos especialistas, chamados de “robôs”, que são aplicados para tentar compor carteiras eficientes de investimentos. Mas, antes de chegar nos modelos e nos algoritmos, é necessário entender e tratar os dados. Nesse sentido, os objetivos desse trabalho são: levantar as fontes de dados de ações, opções e fazer o tratamento delas, para que possam ser utilizadas pelos modelos de otimização e algoritmos especialistas.
